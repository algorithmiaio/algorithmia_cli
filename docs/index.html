Algorithmia CLI

Command line tools for the Algorithmia API

Install it

Linux and MacOS users may simply run the following command in your terminal: 
curl -sSf https://raw.githubusercontent.com/algorithmiaio/algorithmia-cli/master/install.sh | sh

Or see other installation options.

Built for developers

The power of 2500+ algorithms from your terminal

Run any algorithm

$ algo run kenny/factor -d 19635
Completed in 0.0 seconds
[3,5,7,11,17]

$ algo run opencv/SmartThumbnail/2.1.3 -D pinky.jpg -o pinky_mugshot.png
Completed in 13.2 seconds

Manage your data

$ algo mkdir .my/cuteAnimals
Created directory data://.my/labmice

$ algo cp pinky.jpg data://.my/labmice
Uploaded data://.my/labmice/pinky.jpg

Test your algorithms locally

$ algo clone anowell/pinky
Cloning https://git.algorithmia.com/git/anowell/pinky.git
Cloning into 'pinky'...
Username for 'https://git.algorithmia.com': anowell
Password for 'https://anowell@git.algorithmia.com':

$ algo runlocal -d "Are you pondering what I'm pondering?"
Completed in 0.0 seconds
"Narf! I think so, Brain, but what if the Telechubbies don't fight fair?"
